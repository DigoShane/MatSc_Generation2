1. Objective
Train 125M domain-specific LLM for materials science.

2. Architecture
12 layers, 12 heads, 768 hidden, 32k vocab.

3. Dataset Target
20–30M tokens from arXiv materials categories.

4. Hardware
Single RTX 3090 (24GB), local.

5. Training Budget
Max 36 hours wall-clock.

6. Success Metrics
Perplexity improvement + qualitative domain evaluation.


Important Discipline Rule

Do NOT:

1. Increase model size
2. Increase token size beyond 30M
3. Start scraping until this is written
4. Change architecture mid-project
5. Scope discipline = professional signal.
