Abstract

We present TopoTB, a software package written in the Mathematica language, designed to compute electronic struc-
tures, topological properties, and phase diagrams based on tight-binding models. TopoTB is user-friendly, with an inter-
active user interface that enables the tuning of model parameters for fitting the target energy bands in a WYSIWYG way.
In addition, TopoTB also includes functionalities for processing results from Density Functional Theory calculations. The
outputs of TopoTB are rich and readable, and they can be displayed in various styles. These features make TopoTB a
useful tool for the theoretical study of materials.

Keywords: Electronic structure; Topological number; Tight-binding model; k · p model; Interactive operation

1. Introduction

Nontrivial topological states have always been a hot research topic in condensed matter physics, and related materials,
their theoretical models have also been attracting much attention [1, 2, 3, 4, 5, 6]. For instance, the quantum spin Hall
effect in two-dimensional HgTe/CdTe quantum wells [7], the Dirac-type dispersion of surface states in three-dimensional
strong topological insulator Bi2Se3 [8, 9], and so on. These developments often progress in tandem with experiments, first-
principles calculations, and theoretical studies. In experiments, the topological properties are determined by measuring
edge states or surface states using angle-resolved photoemission spectroscopy (ARPES). In first principles calculations,
electronic structure calculations can be performed using DFT code software such as Vienna ab initio simulations package
(VASP) [10, 11], Quantum-Espresso [12], OpenMX [13], ABACUS [14], etc. Then, topological properties can be cal-
culated through the software’s built-in scripts or interfaces with other software. In theoretical studies, for example, the
tight-binding (TB) model, the existence of topology is determined by calculating the Berry phase [15, 16], Berry curvature
[17, 18], Chern number [17], Z2 number [19], etc. These topological related physical quantities require eigenstates, i.e.,
wave functions, which are not unique in quantum mechanics, posing great difficulties for numerical calculations.
There are many software packages available for calculating topological properties. For example, WannierTools [20] is
a open-source software package that requires wannier90_hr.dat files generated by Wannier90 [21], and provides powerful
tools and support for studying the topological properties and phase transitions of materials. Z2pack [22] is a Python
software package that generates the TB models and calculates related topological properties through the Wannier charge
centers (WCCs). PYATB [23] is an efficient Python package for computing electronic structures, topological properties,

as well as other related software packages [24, 25, 26, 27, 28, 29], are excellent tools in specific cases.
Here, we introduce the TopoTB software package, which employs Wilson loop methods and Shiozaki methods with-
out the need for any gauge fixing to compute the topological numbers of the TB model, effectively circumventing the

parameters in the model. We also plan to develop the interfaces with other softwares, such as Wannier90, WannierTools,
etc. For the construction of the model, the MagneticTB [30] and MagneticKP [31] software packages may be helpful. So,
the TopoTB software package focuses on handling the electronic structure and topological properties of the TB model,
and the interface is quite user-friendly.

However, for electronic structure calculations, such as energy bands, density of states (DOS), Fermi surfaces, etc., only

In the TB model, in addition to paying attention to gauges, the basis of the Hamiltonian also needs special attention.
For models considering spin-orbit coupling (SOC), common bases include “orbital grouping” and “spin grouping”. For
“orbital grouping”, the orbital space is placed at the front and the spin space is placed at the back, with the basis represented
as (↑, ↓, ↑, ↓, ...), where ↑, ↓ is the spin index, the orbital index is ignored. For “spin grouping”, the spin space is placed at
the front and the orbital space is placed at the back, with the basis represented as (↑, ↑, ..., ↓, ↓, ...), where ↑, ↓ is the spin
index, the orbital index is ignored.

2.4. Z2 number calculation
There are several methods for calculating the Z2 number, including Pfaffian method [40, 19, 41], Wannier function
center (WFC) method [42, 43, 22, 44], and Berry connection and Berry curvature method [45, 46, 29, 47]. For numerical
calculations, the Fukui-Hatsugai method [46] is a direct method, but this method requires gauge fix on the boundary,
which means it needs to consider translational symmetry, time reversal symmetry, and Kramar degenerates. The WFC
method is an indirect method that does not require any gauges, and the method from Ref.[44] is used in TopoTB software
package. In addition to these, the Shiozaki method [48], which does not require any gauges, is also adopted by TopoTB
software package.

2.4.3. 3D Brillouin zone
In a three-dimensional system, there are 8 time reversal invariant points and 6 time reversal invariant planes, labeled
as (x0, xπ, y0, yπ, z0, zπ). The Z2 number of these 6 planes satisfies

x0 + xπ = y0 + yπ = z0 + zπ (mod 2)
(36)

Therefore, only 4 invariants become independent. We define the Z2 number of the 3D system as

Z2 = (ν0, ν1, ν2, ν3) = (x0 + xπ, xπ, yπ, zπ)
(37)

Specifically, when v0 = 1, the system is called a strong topological insulator [41, 49, 52, 50, 13].

3. Capabilities of TopoTB

The main capabilities of the TopoTB software package are electronic structure and topological properties calculation.
In addition, it can also provide a schematic diagram of the Brillouin zone. Specifically, for two-dimensional lattices, it can
provide the coordinates of the high symmetry point in the first Brillouin zone, which is particularly useful for irregular two-
dimensional lattices. The high symmetry point coordinates it provides can assist WannierTools [20] in calculating surface
states, and so on. The TopoTB software package requires the model Hamiltonian, which can be obtained by extracting
the results (wannier90_hr.dat file) calculated by Wannier90 [21] using our software package, or by using software such
as MagneticTB [30] and MagneticKP [31] software packages. Of course, the TB model can also be built by oneself, and
our software package can also accept such Hamiltonians as inputs. The band structure of the TB model may need to be
compared with the band structure obtained from DFT codes such as VASP [10, 11]. At present, our software package
can process the band data of VASP or the band data obtained from VASPKIT [53] post-processing. In addition to the
conventional band structure, the spin polarized band structure under SOC can also be obtained using our software package.
Specifically, our software package also includes numerous post-processing codes, allowing for interactive operation of the
TB model to determine parameters, compute phase diagrams, and so forth. The main capabilities and possible workflow
of the TopoTB software package are shown in Fig.2.

4. Installation and usage

4.1. Installation
To use the TopoTB software package, you must specify the full path to load the software package via Get or Needs.

5.3. Chern number and Z2 number
5.3.1. Z2 number
In the TopoTB software package, we implemented the Wilson loop method [44] and Shiozaki method [48] to calculate
the Z2 number, which correspond to indirect and direct calculations, respectively. The Z2 number phase diagrams of
the Kane-Mele model were calculated using these two methods, i.e, Z2NumberCalcByWilsonLoop and Z2NumberCalc,
corresponding to Fig.6(a) and Fig.6(b), respectively. In Fig.4(d), we calculated the band gap phase diagram of the Kane-
Mele model and combined it with the Z2 number phase diagram here. It can be seen that the two phase diagrams are
different at the boundary. The relevant code for calculating the Z2 number phase diagram using the Wilson loop method
is provided in Appendix C.1. The relevant code for calculating the Z2 number phase diagrams using the Shiozaki method
is provided in Appendix C.2.
To implement the Shiozaki method, we need to provide the time reversal operator. For the Kane-Mele model, the
corresponding the time reversal operator is T = σ0 ⊗ iσyK, where σi is the Pauli matrix, so the UT in Equation(29) is

1 (*Time reversal operator*)

2 UT=KroneckerProduct[PauliMatrix[0],I*PauliMatrix[2]]

The Hamiltonian is H, and the time reversal symmetry is verified through Equation(30). The corresponding code is

1 ComplexExpand[ConjugateTranspose[UT].Conjugate[H/.{kx->-kx,ky->-ky}].UT]

Referring to Ref.[19], take t = 1 and λS O = 0.06, and the Z2 number phase diagram is a function of λv and λR. Taking
coordinates (1,1) in the phase diagram, i.e., λv = 0.06 and λR = 0.06, both methods yield topologically nontrivial results.
We see that the reference line in Fig.6(c) intersects with the evolution line of the Wannier centers, indicating that the Z2
number is 1. Fig.6(d) shows the edge states corresponding to Fig.6(c), which is consistent with its results. However,
taking coordinates (1,4) in the phase diagram, i.e., λv = 0.06 and λR = 0.24, the results obtained by the two methods are
different. For the Wilson loop method, the evolution line of the Wannier centers obtained is shown in Fig.6(e), with two
intersection points between the reference line and it, corresponding to topologically trivial. The edge states obtained from
this set of parameters are shown in Fig.6(f), and it can be seen that there are edge states. Therefore, near the band gap, the
Wilson loop method may encounter issues, while the Shiozaki method is a better approach.

5.5. Post-processing capabilities

exhibits spin polarization. When considering SOC, its band structure exhibits valley polarization, and by performing S z
spin projection on the band structure, it can be seen that the conduction and valence bands are fully spin polarized. Next,
we will use the TopoTB software package to calculate the band structure and spin projection considering SOC.

6. Conclusion

The TopoTB software package is a user-friendly software with simple input and convenient output and visualization
of data. This software is mainly used to calculate the electronic structure and topological properties of the TB model. In
addition, it also interfaces with other software such as VASP, Wannier90, WannierTools, etc. Through some examples,
we have demonstrated the main capabilities of software and will continuously improve and expand other capabilities. We
hope that this software can become an efficient tool for studying the TB model.
